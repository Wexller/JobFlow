# Work Item Spec Template

Use this tracked template for local work item specs stored under
`docs/workitems/<ID>.md`.

After merge to `main`, move the completed local spec to
`docs/workitems/done/<ID>.md`.

## Minimum Template

```md
# <ID> Title

## Summary

## Goals

## Scope

## Non-Goals

## Affected Areas

## Acceptance Criteria

## Risks

## Verification
```
